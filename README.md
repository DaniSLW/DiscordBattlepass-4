## 🧠 Project Details

This project represents a **Battle Pass interface template**, designed as a structured progression and reward system with a clear and intuitive flow.

It focuses on clean UI structure, tier visibility, and user engagement through a game-style progression model.

---

### 🎮 Overview

- Tier-based reward system with **31 levels**
- Each tier unlocks a **virtual reward**
- Visual progress bar displaying **current tier and completion status**
- Structured to simulate a **modern game progression system**

---

### 🧩 UI & Layout

- Dark minimalist theme with subtle highlight accents
- Grid-based tier layout for structured visibility
- Active tier is visually emphasized
- Locked tiers are clearly marked
- Simple sidebar layout for profile and navigation sections

---

### 🧠 UX Considerations
- Clear hierarchy between active, claimable, and locked rewards
- Simple call-to-action buttons for claiming rewards
- Progress feedback improves user engagement
- Clean spacing and contrast for readability

---

### 🛠️ Technologies Used

- 🧱 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6)

👉 Design and layout fully created without external templates.
